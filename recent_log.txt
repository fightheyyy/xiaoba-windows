
### 实用示例

```typescript
// 提取 Promise 的值类型
type Awaited<T> = T extends Promise<infer U> ? U : T;

type A = Awaited<Promise<string>>; // string

// 深度 Readonly
type DeepReadonly<T> = {
  readonly [P in keyof T]: T[P] extends object
    ? DeepReadonly<T[P]>
    : T[P];
};
```

type World = "world";
type Greeting = `hello ${World}`; // "hello world"

type EmailTemplate = `${EmailLocale}_${EmailType}`;
// "en_welcome" | "en_reset" | "zh_welcome" | "zh_reset"

## 9. 实战技巧

### 类型收窄

```typescript
function process(value: string | number | null) {
  if (value === null) {
    return;
  }
  if (typeof value === "string") {
    return value.toUpperCase();
  }
  return value.toFixed(2);
}
```

### 可辨识联合

```typescript
interface Square {
  kind: "square";
  size: number;
}

interface Circle {
  kind: "circle";
  radius: number;
}

type Shape = Square | Circle;

function area(shape: Shape) {
  switch (shape.kind) {
    case "square":
      return shape.size ** 2;
    case "circle":
      return Math.PI * shape.radius ** 2;
  }
}
```

### 索引访问类型

```typescript
type Person = {
  name: string;
  age: number;
  address: {
    city: string;
  };
};

type Age = Person["age"]; // number
type City = Person["address"]["city"]; // string
```

### typeof 和 keyof

```typescript
const config = {
  host: "localhost",
  port: 3000
};

type Config = typeof config;
// { host: string; port: number; }

type ConfigKeys = keyof Config; // "host" | "port"
```

## 10. 最佳实践

1. **优先使用 interface 定义对象结构**（可扩展）
2. **使用 type 定义联合类型、元组、工具类型**
3. **避免过度使用 any**，用 unknown 替代
4. **善用类型守卫收窄类型**
5. **泛型约束提高类型安全**
6. **利用工具类型减少重复代码**
7. **使用 const 断言保留字面量类型**

```typescript
const colors = ["red", "green"] as const;
type Color = typeof colors[number]; // "red" | "green"
```

## 总结

TypeScript 类型系统核心：
- **基础类型**：原始类型、数组、元组、枚举
- **高级类型**：联合、交叉、字面量、类型守卫
- **泛型**：类型参数化，提高复用性
- **工具类型**：Partial、Pick、Omit 等内置工具
- **映射类型**：批量转换对象类型
- **条件类型**：类型级别的 if-else
- **模板字面量**：字符串类型操作
