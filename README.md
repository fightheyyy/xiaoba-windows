# XiaoBa Windows

XiaoBa 的 Windows 桌面版本 - 基于 Electron 的 AI 助手。

## 功能特性

- 🖥️ **桌面应用** - Electron 打包，原生 Windows 体验
- 🎯 **Dashboard** - 可视化管理界面
- 🔌 **Skill 系统** - 可扩展的能力插件
- 🏪 **Skill 商店** - 从官方 Hub 安装社区 Skills

## 快速开始

下载 `XiaoBa Setup 0.1.0.exe` 并安装。

## 开发

```bash
npm install
npm run electron:dev
npm run electron:build:win
```
